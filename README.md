super_unko.vim
==============

世界一クリーンなプロジェクトのVimPlugin版です。

Installation
------------

### vim-plug

[vim-plug](https://github.com/junegunn/vim-plug)

```vim
call plug#begin('~/.vim/plugged')

Plug 'unkontributors/super_unko.vim'

call plug#end()
```

Usage
-----

Call below command. Set a 💩 to current line.

```vim
:UnkoTower
:UnkoTower 5
```
