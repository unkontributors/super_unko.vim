Installation
------------

### vim-plug

[vim-plug](https://github.com/junegunn/vim-plug)

Add below settings.

```vim
call plug#begin('~/.vim/plugged')

Plug 'unkontributors/super_unko.vim'

call plug#end()
```

Usage
-----

Call below command. Set a ğŸ’© to current line.


